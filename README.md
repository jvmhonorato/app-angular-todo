## Angular TO-DO-LIST  (CRUD)


application to manage TASKS


## Technology 

Here are the technologies used in this project.

* Angular
* SCSS
* Typescript




## Services Used

* Github





## Getting started

* Dependency
 - Angular forms
 - Karma Jasmine
 - Angular routes
 
  
  
* To install the dependencies.
  - npm install
  
* To run the project.
  - ng server
